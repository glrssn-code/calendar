# 日历应用升级计划

## 后续功能清单

### 高优先级
- [ ] **月视图** — 很多人习惯看月历，比周视图更直观
- [ ] **重复事件** — 每周一开会、每天吃药等自动重复
- [ ] **导入导出增强** — 支持 CSV/Excel 导入、ical 格式同步
- [ ] **键盘快捷键** — 快速导航、快捷创建，提升效率

### 中优先级
- [ ] **深色模式** — 夜间使用更护眼（目前只有主题切换）
- [ ] **桌面通知** — 不仅是浏览器通知，支持系统通知
- [ ] **搜索功能增强** — 快速找到历史事件和便签
- [ ] **多日历切换** — 工作/生活分开管理

### 锦上添花
- [ ] **拖拽调整时长** — 直接拖拽事件边缘调整长度
- [ ] **事件复制** — 复制已有事件快速创建
- [ ] **农历显示** — 中国特色需求
- [ ] **天气插件** — 在日历上显示天气
